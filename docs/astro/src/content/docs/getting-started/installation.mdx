### Requirements

ZFS CLI installed on your system and accessible from your PATH.

### Release channels

There are currently two release channels:

- [beta](https://github.com/backup-blob/zfs-backup-blob/tree/beta) - "beta" releases
- [next](https://github.com/backup-blob/zfs-backup-blob/tree/next) - nightly builds

We recommend using the beta channel for normal users.

### Linux

1. Download the latest release from the <a href="https://github.com/backup-blob/zfs-backup-blob/releases" target="_blank">releases page</a> (pick a beta release)
2. Extract the binary
3. Move the binary to a location in your PATH

### MacOS

1. Download the latest release from the <a href="https://github.com/backup-blob/zfs-backup-blob/releases" target="_blank">releases page</a> (pick a beta release)
2. Extract the binary
3. Move the binary to a location in your PATH
